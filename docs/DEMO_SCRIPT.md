# Demo Script

## Demo 1: Paper Flow

1. Open the Android shell.
2. Show `Dashboard`.
3. Go to `Discover`.
4. Enter `coffee cat`, `0.02 SOL`, `normal`.
5. Copy the recommended contract into `Review`.
6. Execute in `paper` mode.
7. Open `Positions` and `PnL`.

## Demo 2: Auto Flow

1. Open the `Auto` tab.
2. Start auto with `coffee cat` and `0.02 SOL`.
3. Show running status, last action, and latest PnL.
4. If rate-limited or paused, show the pause reason.
5. Stop auto cleanly.

## Demo 3: Emulator Dev Fallback

1. Start backend on macOS.
2. Use `10.0.2.2`.
3. Show that the same UI works from the emulator.
