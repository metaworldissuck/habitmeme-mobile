# HabitMeme Mobile

HabitMeme Mobile is the final submission build for the hackathon.

## What it includes

- Android WebView shell
- local FastAPI backend for Termux
- paper, semi-auto, and auto trading modes
- SQLite-based trade, position, and PnL tracking
- 429, timeout, and circuit-breaker protections

## What stays unchanged

The original `bitget-wallet-skill` directory remains untouched as the legacy Python CLI implementation.
